A ViM setup
===========

Contains a complete ViM setup including bundles and a powerline-enabled
monospace font.

Bundles
-------

  * Ada-Bundle
  * SingleCompile
  * YankRing.vim
  * clang_complete
  * ctrlp.vim
  * gitv
  * gundo.vim
  * jedi-vim
  * nerdtree
  * snipmate-snippets
  * supertab
  * taglist.vim
  * tlib_vim
  * vim
  * vim-addon-mw-utils
  * vim-colors-solarized
  * vim-five
  * vim-flake8
  * vim-javascript
  * vim-nerdtree-tabs
  * vim-orgmode
  * vim-pathogen
  * vim-powerline
  * vim-snipmate

Installing
----------

You may want to make a backup of your .vim directory first.
To install, just do

    mkdir ~/.vimrc
    cd Dotfiles/vim
    bash install.sh
